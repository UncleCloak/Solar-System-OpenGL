# 3D Solar System in OpenGL
### A Computer Graphics Project
![C++](https://img.shields.io/badge/c++-%2300599C.svg?style=for-the-badge&logo=c%2B%2B&logoColor=white) ![OpenGL](https://img.shields.io/badge/OpenGL-%23FFFFFF.svg?style=for-the-badge&logo=opengl) <br>
This project aims to simulate an animated solar system on your computer, coded entirely in C++ and using OpenGL, this project includes the following:
  - The Sun
  - All Planets (No Pluto)
  - Accurate Planet Spins (Days) And Orbits (Years)
  - Accurate Colors (2 Color Gradient) And Distances (From The Center, aka The Sun)
  - A Galaxy Background Picture
  - A Movable Camera
